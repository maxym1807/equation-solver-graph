# Решатель уравнений + Построитель графиков

**Интерактивная программа** для решения уравнений и мгновенного построения графиков.

![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)
![Tkinter](https://img.shields.io/badge/Tkinter-GUI-green)
![SymPy](https://img.shields.io/badge/SymPy-Symbolic-orange)
![Matplotlib](https://img.shields.io/badge/Matplotlib-Graphs-red)

## Возможности

- Решение линейных и квадратичных уравнений
- Автоматическое построение графика функции
- Поиск и отображение корней на графике
- Удобный графический интерфейс (Tkinter)
- Красивые и понятные графики

## Примеры работы

**Ввод:** `x**2 - 5*x + 6 = 0`  
**Результат:** `x = 2`, `x = 3` + график параболы с отмеченными корнями

**Ввод:** `x**2 = 16`  
**Результат:** `x = -4`, `x = 4` + график
