# Розв’язувач рівнянь + Конструктор графіків

**Інтерактивна програма** для розв’язання рівнянь та миттєвої побудови графіків.

![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)
![Tkinter](https://img.shields.io/badge/Tkinter-GUI-green)
![SymPy](https://img.shields.io/badge/SymPy-Symbolic-orange)
![Matplotlib](https://img.shields.io/badge/Matplotlib-Graphs-red)

## Можливості

- Розв'язання лінійних та квадратичних рівнянь
- Пошук та відображення коренів на графіку
- Зручний графічний інтерфейс (Tkinter)
- Красиві та зрозумілі графіки

## Приклади роботи

**Вхідні дані:** `x**2 = 25`  
**Результат:** `x = -5`, `x = 5` + графік параболи з позначеними коренями

**Введення:** `x**2 - x - 30 = 0`  
**Результат:** `x = -5`, `x = 6` + графік параболи з позначеними коренями
